Configurable Products Settings¶

In the system configuration, you can control the way a matrix form for a configurable product is displayed in the storefront. Configurable product information can be set on three levels: globally, per organization, and per website. Website settings override organization, organization settings override the system ones.

  • Customize Configurable Products Globally
  • Customize Configurable Products Settings per Organization
  • Customize Configurable Products Settings per Website

Customize Configurable Products Globally¶

  1. Navigate to System > Configuration in the main menu.
  2. Select Commerce > Product > Configurable Products in the menu to the left.

Note

For faster navigation between the configuration menu sections, use Quick Search.

The following page opens:

Here, you can customize a matrix form view on a product page:

  1. Clear the Use Default check box next to Display Matrix Form.

  2. Select the necessary option from the following list:

    • Do Not Display — The option disables the matrix form visibility for a configurable product and displays an alternative table with a possibility to select one simple product within a configurable one at a time.

    • Inline Matrix Form — With this option enabled, the matrix form is displayed next to the product image on the product details page.

    • Popup Matrix Form — The option triggers a popup matrix form upon clicking the Update Shopping List button. Once the popup displays, a customer can select multiple simple products at a time.

  3. Click Save Settings.

Customize Configurable Products Settings per Organization¶

To change the default settings for an organization:

  1. Navigate to **System > User Management > Organization** in the main menu.
  2. For the necessary organization, hover over the More Options menu to the right and click to start editing the configuration.
  3. Select Commerce > Product > Configurable Products in the menu to the left.

Note

For faster navigation between the configuration menu sections, use Quick Search.

To change the default settings for a website:

  1. Navigate to System > Websites in the main menu.
  2. For the necessary website, hover over the More Options menu to the right and click to start editing the configuration.
  3. Select Commerce > Product > Configurable Products in the menu to the left.
