Configure Upload Settings per OrganizationÂ¶

To configure MIME types per organization:

  1. Navigate to System > User Management > Organizations in the main menu.
  2. For the necessary organization, hover over the More Options menu and click to start editing the configuration.
  3. Select System Configuration > General Setup > Upload Settings in the menu on the left.

Note

For faster navigation between the configuration menu sections, use Quick Search.

  1. The following page opens:

  2. Clear the Use Default check box next to the option.

  3. Select a set of mime types that will be supported for the file and image attachments in the system.

  4. Add any MIME type by writing the required file or image format in the text box.

  5. Click Save Settings.

If attachments are enabled for an entity, the configuration of the entity will prevail and override the corresponding settings configured both globally and per organization.
