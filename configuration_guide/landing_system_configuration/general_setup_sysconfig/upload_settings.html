Upload Settings¶

To define the formats of the files to be uploaded into the website management console or the storefront, you need to configure the corresponding MIME types settings in the system configuration.

MIME types are Multipurpose Internet Mail Extension types which help identify the types of the attachments and, thus, limit the possibility of uploading the documents of inappropriate extensions.

The MIME types settings can be configured globally and per organization.

  • Configure Upload Settings Globally
  • Configure Upload Settings per Organization

Configure Upload Settings Globally¶

To configure MIME types globally:

  1. Navigate to System > Configuration in the main menu.
  2. In the System Configuration menu to the left, expand General Setup and click Upload Settings.

Note

For faster navigation between the configuration menu sections, use Quick Search.

  1. The following page opens:

  2. Clear the Use Default check box next to the option.

  3. Select a set of mime types that will be supported for the file and image attachments in the system.

  4. Add any MIME type by writing the required file or image format in the text box.

  5. Click Save Settings.

If attachments are enabled for an entity, the configuration of the entity will prevail and override the corresponding global one.

Configure Upload Settings per Organization¶

To configure MIME types per organization:

  1. Navigate to System > User Management > Organizations in the main menu.
  2. For the necessary organization, hover over the More Options menu and click to start editing the configuration.
  3. Select System Configuration > General Setup > Upload Settings in the menu on the left.

Note

For faster navigation between the configuration menu sections, use Quick Search.

  1. The following page opens:

  2. Clear the Use Default check box next to the option.

  3. Select a set of mime types that will be supported for the file and image attachments in the system.

  4. Add any MIME type by writing the required file or image format in the text box.

  5. Click Save Settings.

If attachments are enabled for an entity, the configuration of the entity will prevail and override the corresponding settings configured both globally and per organization.
