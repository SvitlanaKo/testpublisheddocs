To create a new Customer:

  1. Navigate to Customers > Customer Users in the main menu.

  2. Click Create Customer User.

    The following page opens:

  3. Fill in customer Name and other personal information.

  4. Select a customer this user represents.

  5. If you are adding a subsidiary of the existing customer, select Parent Customer.

  6. Assign a sales representative who will be assisting this customer user. By default, the customer sales representative applies to the customer user.

  7. Check the Generate Password and Send Welcome Email boxes.

  8. Select the Website the customer user will be redirected to upon the login. See Managing Websites for more information.

  9. Add billing and shipping address as described in Address Book section.

  10. In the Roles section, select the roles that should apply to the customer user. When several roles are selected, granted permissions are accumulated from all the assigned roles. See Managing Customer User Roles for more information.

  11. Click Save on the top right of the page.

The new Customer User is created.
