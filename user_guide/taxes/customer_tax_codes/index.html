Customer Tax Codes¶

Customer Tax Code is a label that is assigned to a customer and indicates the tax obligations and exemptions a customer has. The customer tax obligations are taken into account when a customer (user) submits an order.

The sections below provide guidance on managing customer tax codes, using them to label customers and bind customers to a dedicated tax rules.

  • Create a Customer Tax Code
  • View Customer Tax Code Details, Linked Customer Accounts and Change History
  • Edit Customer Tax Code Details
    • Association of the Customer Tax Code with the Customer
    • Association of the Customer Tax Code with the Tax Rule

View and Filter Customer Tax Codes¶

To view all Customer Tax Codes, navigate to Taxes > Customer Tax Codes in the main menu.

Note

To handle big volume of data, use page switcher, increase View Per Page or use filters to narrow down the list to just the codes you need.

The following information about the Customer Tax Codes is available in the Customer Tax Codes list:

Name Description
CODE The unique identifier of the Customer Tax Code.
DESCRIPTION Detailed description of the Customer Tax Code.
CREATED AT Date when the Customer Tax Code was created.
UPDATED AT Date when the Customer Tax Code was updated last.

Next steps

You can perform the following actions with every item in the Customer Tax Codes list:

  • View a Customer Tax Code details: Click on the item to open its details page.

    OR

    Aternatively: Hover over the more actions menu to the right of the item and click the to open its details page.

  • Edit a Customer Tax Code details: Hover over the more actions menu to the right of the item and click the to start editing its details.

  • Delete a Customer Tax Code: Hover over the more actions menu to the right of the item and click the to remove the Customer Tax Code.

  • Create Account Tax Code: Click Create Account Tax Code to the top right from the list.

Create a Customer Tax Code¶

To create a new Customer Tax Code:

  1. Navigate to Taxes > Customer Tax Codes in the main menu.

  2. Click Create Customer Tax Codes.

    The following page opens:

  3. Fill in Code and Description with information about the Customer Tax Code you are creating.

  4. Click Save on the top right of the page.

The new Customer Tax Code is created.

View Customer Tax Code Details, Linked Customer Accounts and Change History¶

To view a Customer Tax Code details:

  1. Navigate to Taxes > Customer Tax Codes in the main menu.

  2. Find the line with the necessary Customer Tax Code and click on it.

    The following information is available immediately:

    • Code
    • Decription
    • List of customers with this customer tax code
    • List of customer groups with this customer tax code

Next steps

You can perform the following actions with a Customer Tax Code:

  • View Change History: See the log of edits of this customer tax code by clicking on the Change History link to the tp right of the page.

  • View details of the Customer who is linked to this customer tax code: Click on the line with the customer account details.

    Note

    You can edit the customer details to remove link to this customer tax code, if necessary.

  • Edit a Customer Tax Code details: Click Edit to the top right of the page.

  • Delete a Customer Tax Code: Click Delete to the top right of the page.

Edit Customer Tax Code Details¶

To edit the Customer Tax Code value and description:

  1. Navigate to Taxes > Customer Tax Codes in the main menu.

  2. Hower over the more actions menu to the right of the item and click the to start editing its details. The following page opens:

  3. Update the Code and Description with new information about the Customer Tax Code.

  4. Click Save on the top right of the page.

The Customer Tax Code is updated.

Association of the Customer Tax Code with the Customer¶

You can edit the association of the Customer Tax Code with the customer when editing the customer details.

See information on linking a tax code to a customer for more information.

Association of the Customer Tax Code with the Tax Rule¶

You can edit the association of the Customer Tax Code with the tax when editing the tax rule details (see the respective topic for more information).
