View Customer Tax Code Details, Linked Customer Accounts and Change HistoryÂ¶

To view a Customer Tax Code details:

  1. Navigate to Taxes > Customer Tax Codes in the main menu.

  2. Find the line with the necessary Customer Tax Code and click on it.

    The following information is available immediately:

    • Code
    • Decription
    • List of customers with this customer tax code
    • List of customer groups with this customer tax code

Next steps

You can perform the following actions with a Customer Tax Code:

  • View Change History: See the log of edits of this customer tax code by clicking on the Change History link to the tp right of the page.

  • View details of the Customer who is linked to this customer tax code: Click on the line with the customer account details.

    Note

    You can edit the customer details to remove link to this customer tax code, if necessary.

  • Edit a Customer Tax Code details: Click Edit to the top right of the page.

  • Delete a Customer Tax Code: Click Delete to the top right of the page.
