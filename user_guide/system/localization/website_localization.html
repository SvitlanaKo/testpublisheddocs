To define the custom localization options for the particular website:

  1. Navigate to System > Websites using the main menu.

  2. Click on the website you would like to customize localization settings for (e.g. Australia).

  3. On the website details page, click Configuration.

  4. In the System Configuration menu to the left, expand General Setup and click Localization.

    The following page opens:

  5. Select the localizations to be enabled for this website. Use the Ctrl and Shift keys and click to choose the languages from the list.

  6. Select the default localization for this website. Localization affects formatting of numbers, addresses, names, and dates.
