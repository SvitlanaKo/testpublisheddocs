Add a Context to an Existing Calendar Event¶

From the Calendar Event View Page¶

To add a context to an existing calendar event when viewing the event details:

  1. In the main menu, navigate to Activities>Calendar Events.

  2. In the grid of the Calendar Events page, click the required event.

    Alternatively, you can click the ellipsis menu at the end of the corresponding row and then click the View icon.

  3. On the calendar event view page, click the Add Context button in the upper-right corner of the page.

  4. In the Add Context Entity dialog box, select an entity of the related (context) record from the list in the upper-left corner of the dialog box.



  5. In the main grid of the dialog box, click the required record.

    The context record is added and is shown on the event view page under the calendar event name.


From the Related Entity View Page¶

Entity records that are specified in a calendar event context have the event displayed in the Activity sections of their view pages.

Add more records as a context to a calendar event directly from one of the related record view page using the following steps:

  1. On the entity record view page, click Activity.

  2. In the Activity section, click the ellipsis menu at the end of the row with the event you want to add context to and then click Add Context.



  1. In the Add Context Entity dialog box, select an entity of the related (context) record from the list in the upper-left corner of the dialog box.



  2. In the main grid of the dialog box, click the required record.

    The context record is added and is shown in the calendar event details.



From the My Calendar page or Today’s Calendar Widget¶

To add a context to an existing calendar event from a calendar (in Today’s calendar widget or in your calendar page):

  1. Click the event in the calendar.

  2. On the View Event calendar card, click Edit.

  3. In the context field, start typing the name of the record that should be linked to the event as a context. The list of suggestions shows up automatically and is dynamically filtered as you type. Click on the suggested item to add it to a context. Alternatively, to add the highlighted suggested item, press Enter.



Remove a Context from a Calendar Event¶

From the Calendar Event View Page¶

To delete a calendar event’s context when viewing the event details:

  1. In the main menu, navigate to Activities>Calendar Events.

  2. In the grid of the Calendar Events page, click the required event.

    Alternatively, you can click the ellipsis menu at the end of the corresponding row and then click the View icon.

  3. On the calendar event view page, under the calendar event name in the upper-left corner of the page, click the x icon next to the context that you want to remove.



From the Related Entity View Page¶

Entity records that are specified in the calendar event context have this call displayed in the Activity sections of their view pages.

To remove a calendar event’s context directly from the related entity view page:

  1. On the entity record view page, click Activity.

  2. In the Activity section, click the calendar event that you want to remove context from.

    The event details open.

  3. In the event details, click the x icon next to the context that you would like to remove.



From the My Calendar page / Today’s Calendar Widget¶

To delete a calendar event’s context from the calendar in Today’s calendar widget or in your calendar page:

  1. Click the event in the calendar.

  2. On the View Event calendar card, click the x icon next to the context that you would like to remove.
